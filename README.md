# 🎓 Smriti 5.0 | Civil Engineering Farewell Invitation

A premium, cinematic single-page web invitation designed for the Civil Engineering batch farewell. Built with a modern architectural aesthetic, this project features deep slate backgrounds, metallic gold accents, glassmorphism UI cards, and an interactive scroll-to-play video core.

## ✨ Features

*   **🎥 Scroll-to-Play Video:** Utilizes the JavaScript `IntersectionObserver` API to automatically trigger video playback the moment the user scrolls it into the viewport.
*   **🔊 Interactive Unmute System:** Features a custom "Tap for Sound" overlay button to elegantly bypass modern browser autoplay restrictions while keeping the user engaged.
*   **💅 Premium UI/UX:** Designed with a "Cinematic Engineering" theme, featuring floating background light animations, dynamic glowing text, and frosted-glass effect panels.
*   **📱 Fully Responsive:** Strict mobile width control ensures the video player and glass panels scale perfectly on any device without horizontal scrolling.
*   **🏗️ Zero Build Tools Required:** Styled entirely using Tailwind CSS via CDN for rapid deployment and easy editing.
